# ☕ Café du Coin

Application web de gestion de commandes pour un café, développée en Python avec Flask.

## Ce que fait l'application
- Passer et gérer des commandes (café, thé, croissants...)
- Système de connexion sécurisé
- Export des commandes en CSV
- Interface web accessible depuis un navigateur

## Technologies utilisées
- Python / Flask
- SQLite
- HTML / CSS / Jinja2
- PyInstaller (packaging en .exe)

## État du projet
✅ Étapes 1 à 8 terminées (application complète et fonctionnelle)
⬜ Étape 8 Bis (icône personnalisée et splash screen) — en cours

Ce projet a été entièrement conçu et développé avec l'aide de l'IA GitHub Copilot.
Je suis débutant en programmation, sans formation payante, sans école de code.
Juste moi, ma curiosité, et une IA pour m'accompagner étape par étape.

d'euros en formations qui fleurissent partout sur internet.
